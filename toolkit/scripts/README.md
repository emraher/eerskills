# Toolkit Scripts

This directory contains automated detection and cleanup tools.

## Scripts

- **detect_slop.py** - Wraps the Python detection script from `external/cc-polymath`.
- **clean_slop.py** - Wraps the Python cleanup script from `external/cc-polymath`.
- **detect_slop.R** - Native R implementation of slop detection (unique to this repo).
